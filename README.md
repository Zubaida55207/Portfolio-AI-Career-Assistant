# 🌐 Zubaida’s Portfolio Website + AI Career Assistant 🤖

A modern, interactive **portfolio website** built with **HTML, CSS, and JavaScript**, integrated with an **AI Career Assistant Chatbot** developed using **Python, Gradio, and Machine Learning**.  

Visitors (including recruiters) can **chat with the AI Assistant** to learn about my **skills, projects, career goals, and achievements** in real-time.

---

## ✨ Features

### 💻 Portfolio Website
- Built using **HTML, CSS, and JavaScript**
- Fully responsive dark theme
- Projects, skills, and contact sections
- Integrated **floating chatbot widget**

### 🤖 AI Career Assistant
- Created using **Python + Gradio**
- Answers recruiter-style questions like:
  - “Why should we hire you?”
  - “Tell me about yourself”
  - “What are your strengths?”
- Retrieves information from my local files: Bio, Projects, Goals, and LinkedIn
- Can be embedded easily in any website or portfolio

---

## 🧰 Tech Stack

**Frontend:**  
HTML, CSS, JavaScript  

**Backend / ML:**  
Python, Gradio, Scikit-learn (TF-IDF + Cosine Similarity)
